== Description ==

This is a Gutenberg block that allows to create a Youtube banner, a full width, full height, covering the whole viewport video. It has play and mute button for controlling the video.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== JQuery ==

This Gutenberg block needs JQuery enabled for frontend.
